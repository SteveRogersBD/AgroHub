## Troubleshooting

**Problem:** Weather doesn't load  
**Solution:** Check internet connection and API key

**Problem:** Build error  
**Solution:** Sync Gradle files and rebuild

**Problem:** Wrong location  
**Solution:** Grant location permissions or edit default location
